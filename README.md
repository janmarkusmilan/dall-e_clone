# DALL-E Clone

Clone of DALL-E, an AI image generation tool.

Technologies:

- MERN (MongoDB, Express, React, Node)
- TailwindCSS
- OpenAI Dall-E Model
- Cloudinary (Image storage)

How to run on local machine:

- Terminal 1: cd client -> npm run dev
- Terminal 2: cd server -> npm start
